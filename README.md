# Address-book
Aplikacja Towrzy książkę adresową zapisaną w pliku .db

## 
- Tkinter
- SQLite3
- PIL (Python Imaging Library)

## Zbierane Dane
- Imię
- Drugie Imię
- Nazwisko
- Ulica
- Miasto
- Kod pocztowy
- Numer Telefonu
- Adres Email

## Opcje
- Dodawanie nowych kontaktów
- Usuwanie kontaktów
- Wyświetlanie zapisanych kontaktów
- Edycja kontaktów
